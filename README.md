# 🎮 KBC Mini Project (C++)

A simple console-based **Kaun Banega Crorepati (KBC)** game built using C++.
This project simulates a quiz game with prize money, lifelines, and multiple-choice questions.

---

## 🚀 Features

* 💰 Prize ladder system
* ❓ Multiple-choice questions (2×2 format like KBC)
* 🧠 Lifelines:

  * 50-50
  * Audience Poll
  * Phone a Friend
  * Expert Advice
* 🔒 Each lifeline can be used only once
* 📈 Running total of winning amount
* 🎯 Simple and beginner-friendly code

---

## 🖥️ How It Works

* Player is asked 7 questions
* Each correct answer increases total winning money
* Wrong answer ends the game
* Player can use lifelines for help

---

## 🛠️ Technologies Used

* C++
* Standard Library (`iostream`, `cstdlib`, `ctime`)

---

## ▶️ How to Run

1. Compile the code:

   ```bash
   g++ kbc.cpp -o kbc
   ```

2. Run the program:

   ```bash
   ./kbc
   ```
